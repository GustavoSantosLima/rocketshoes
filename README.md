# ROCKETSHOES

### Pré-requisitos

1. Instale [JSON SERVER](https://github.com/typicode/json-server)

### Começando

2. Clone o repositório e entre na pasta.

   ```
     git clone https://github.com/GustavoSantosLima/rocketshoes.git

     cd rocketshoes
   ```

3. Instale as dependencias do `package.json`.

   ```
     npm install or yarn
   ```
